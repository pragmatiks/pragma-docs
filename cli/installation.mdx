## Requirements

- Python 3.13+

## Install

<Tabs>
  <Tab title="uvx">
    ```bash
    uvx pragmatiks-cli
    ```

    Or install globally:

    ```bash
    uv tool install pragmatiks-cli
    ```
  </Tab>
  <Tab title="pipx">
    ```bash
    pipx install pragmatiks-cli
    ```
  </Tab>
  <Tab title="pip">
    ```bash
    pip install pragmatiks-cli
    ```
  </Tab>
</Tabs>

## Verify

```bash
pragma --version
```

## Shell Completion

<Tabs>
  <Tab title="Bash">
    ```bash
    eval "$(pragma --show-completion bash)"
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    eval "$(pragma --show-completion zsh)"
    ```
  </Tab>
  <Tab title="Fish">
    ```fish
    pragma --show-completion fish | source
    ```
  </Tab>
</Tabs>
